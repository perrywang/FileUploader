# FileUploader
## Description
- Support multiple files upload
- Ajax upload progress indicator
- Easy Usage
- Java spring server example code provided

## Required
- jquery
- semantic-ui
- handlebars

## Usage
```
$(document).ready(function(){
		$('#container').fileUploader({url:'/upload'});
});
```
